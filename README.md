# PIGU
Dépôt PIGU prototype fonctionnel
Le projet P.I.G.U vise à développer une plateforme numérique intelligente d’aide à la
régulation médicale d’urgence pour le SAMU National.

Le prototype doit permettre de :
 simuler la réception d’un appel d’urgence
 visualiser la localisation d’un incident
 afficher les ambulances disponibles
 suggérer une ambulance
 sélectionner un hôpital de destination
 afficher un tableau de bord simple
 démontrer une logique d’aide à la décision
